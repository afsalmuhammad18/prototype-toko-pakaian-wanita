# Rona Rupa — Fashion Perempuan Lokal (Web Prototype)

Rona Rupa adalah sebuah prototype platform *e-commerce* fashion perempuan yang mengusung nilai estetika lokal, ramah lingkungan, dan pemberdayaan pengrajin Indonesia. Prototype ini dibangun dengan fokus pada pengalaman pengguna (*UX*) yang elegan, responsif, dan fungsional.


## 🌿 Tentang Proyek
Proyek ini merupakan demonstrasi aplikasi web satu halaman (Single Page Application) murni menggunakan **HTML5**, **CSS3**, dan **Vanilla JavaScript**. Tanpa bantuan *framework* eksternal (seperti React atau Vue), prototype ini mensimulasikan alur belanja lengkap dari eksplorasi produk hingga proses checkout.

### Fitur Utama:
- **Responsive Design:** Optimal di perangkat mobile (Android 14 Tablet/Smartphone) maupun desktop (Linux CachyOS).
- **SPA Routing:** Perpindahan halaman mulus tanpa *reload* menggunakan navigasi berbasis Hash (`#/`).
- **Manajemen State:** Fitur Keranjang Belanja dan Wishlist yang tersimpan secara lokal (*Local Storage*).
- **Katalog Dinamis:** Filter berdasarkan kategori, ukuran, harga, serta fitur sortir.
- **Detail Produk:** Galeri foto, pemilihan warna/ukuran, panduan ukuran, dan sistem ulasan.
- **Checkout Flow:** Simulasi pengisian data penerima, pemilihan kurir, dan metode pembayaran.

## 🛠️ Spesifikasi Teknis
Prototype ini dikembangkan dengan optimasi untuk spesifikasi berikut:
- **Teknologi:** HTML5 (Semantic), CSS3 (Custom Properties & Grid), Vanilla JavaScript (ES6+).
- **Fonts:** Playfair Display (Serif) & Inter (Sans-serif) via Google Fonts.
- **Icons:** Custom SVG Icons untuk performa ringan.
- **Asset:** Gambar menggunakan Unsplash API (Auto-format & Crop).
